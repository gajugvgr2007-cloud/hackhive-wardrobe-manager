HOW TO USE
================================================================================

1. INSTALL:
   npm install

2. CONFIGURE (.env.local):
   NEXT_PUBLIC_SUPABASE_URL=your_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
   NEXT_PUBLIC_OPENWEATHER_API_KEY=your_key

3. RUN:
   npm run dev

4. VISIT:
   http://localhost:3000

5. EXPLORE:
   • Click sidebar items
   • Expand weather widget
   • Hover for animations
   • Interact with all cards
