FEATURE OVERVIEW
================================================================================

The Digital Wardrobe is a complete clothing management system featuring:
- Professional image upload with live preview
- Responsive masonry grid layout (1-4 columns)
- Intelligent occasion tagging (Formal, Casual, Workout)
- Intelligent weather tagging (Hot, Cold, Rainy)
- Beautiful glowing badge overlays
- Smooth Framer Motion animations
- Real-time filtering by category
- Delete functionality with instant updates

================================================================================
DATA STRUCTURE
================================================================================

ClothingItem Interface:
  {
    id: string,                    // Unique identifier (timestamp)
    name: string,                  // Item display name
    category: string,              // Tops, Bottoms, Shoes, etc.
    image_url: string,             // Base64 encoded image or URL
    occasions: string[],           // ["Formal", "Casual", "Workout"]
    weather_types: string[],       // ["Hot", "Cold", "Rainy"]
    created_at?: string            // ISO timestamp string
  }

FormData Interface:
  {
    name: string,
    category: string,
    occasions: string[],
    weather_types: string[],
    image: File | null
  }
